Se requiere migrar una app de vfp a web.

 Desarrollo en máximo 2 semanas de un sistema web para realizar solicitudes de un inventario.

Una ventana
Con el Login usuario y contraseña (dependiendo de a que departamento está asociado se le muestran ciertos artículos, puede tener 2 o más departamentos)

Otra ventana
En el cual se pueda muestre los artículos dependiendo del tipo de usuario que inicia sesión. Y permita buscar por código o nombre y escoger un artículo para añadirlo a la solicitud donde también permita escoger la cantidad y también al momento de hacer la solicitud se mande hacia una base de datos el registro para tener constancia y le permita imprimir esa solicitud para tener constancia física.

Otra ventana que permita modificar los productos. Y una opción que permita deshabilitar o habilitar estos mismos y que aparezca.

Total 3 ventanas.
